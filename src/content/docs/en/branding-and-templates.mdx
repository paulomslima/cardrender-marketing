Your workspace branding is applied across cards, email signatures, and the team directory. Update once and watch the entire team stay consistent.

## Workspace Branding

Centralized branding ensures every card reflects your company identity without requiring individual customization.

### Brand Assets

Upload and manage core brand elements:

**Logo**
- Primary logo (PNG, SVG recommended)
- Light and dark variants for different backgrounds
- Favicon for branded directories
- Maximum file size: 2MB

**Colors**
- Primary brand color (used for accents and CTAs)
- Secondary color (for highlights and gradients)
- Text colors (heading and body)
- Background colors (light and dark modes)

Supports hex, RGB, or HSL values. Color picker includes accessibility contrast checker.

### Brand Messaging

Define default content for team directory:
- Company tagline or value proposition
- About section (supports markdown)
- Social media links
- Contact information

## Card Templates

Templates control layout, typography, and which fields are editable by team members.

### Template Library

Choose from pre-built templates:

**Professional**
- Clean, minimal design
- Focus on contact information
- Ideal for corporate environments

**Creative**
- Bold colors and custom layouts
- Photo-first design
- Great for agencies and startups

**Executive**
- Sophisticated, understated design
- Emphasis on credentials and bio
- Perfect for C-suite and board members

**Sales**
- CTA-focused with booking links
- Social proof and testimonials
- Optimized for conversion

### Custom Templates

Premium and Enterprise plans can create custom templates:

1. Start with a base template or blank canvas
2. Arrange layout sections (header, contact, bio, links, footer)
3. Choose typography and spacing
4. Set default colors and images
5. Define editable vs. locked fields
6. Preview across devices
7. Publish to team

### Field Locking

Control which template fields members can customize:

**Locked fields** (enforced across team):
- Company logo and colors
- Header layout and background
- Footer content (legal disclaimers, company info)
- Social media icon styles

**Editable fields** (member customizable):
- Name and title
- Photo or avatar
- Personal contact information
- Bio and description
- Custom links and CTAs

## Email Signatures

Generate branded email signatures that match your card templates.

### Signature Builder

1. Select card template as base
2. Choose signature layout (horizontal, vertical, compact)
3. Add optional elements:
   - Profile photo
   - Social media icons
   - Company logo
   - Legal disclaimers
   - Banner images
4. Export for email clients (Outlook, Gmail, Apple Mail)

### Distribution

**Manual installation:**
- Members copy HTML signature code
- Paste into email client settings

**Automated deployment (Enterprise):**
- Google Workspace integration
- Microsoft 365 integration
- Signature management platforms (Exclaimer, CodeTwo)

## Team Directory Customization

The public directory showcases all team members in one branded page.

### Layout Options

**Grid View**
- Card-style layout
- Best for visual browsing
- Shows photos prominently

**List View**
- Compact, scannable format
- Ideal for large teams
- Sortable by name, role, location

**Masonry**
- Pinterest-style layout
- Dynamic, engaging design
- Works well with varied photo sizes

### Directory Settings

- **URL slug:** Choose custom subdirectory (e.g., `/team` or `/directory`)
- **Header banner:** Add hero image or video
- **Search & filters:** Enable by role, team, location, or department
- **SEO metadata:** Custom title, description, and social preview
- **Custom CSS:** Enterprise plans can inject custom styles

## Brand Consistency Enforcement

### Approval Workflows

Require admin approval for:
- New card publications
- Template changes
- Brand asset updates
- Directory modifications

### Audit & Monitoring

Track brand compliance:
- Flag cards using off-brand colors or fonts
- Identify outdated logos or contact information
- Monitor template adoption rates
- Generate brand compliance reports

## Design System Integration

Enterprise customers can sync with existing design systems:

**Figma Integration**
- Import design tokens (colors, typography, spacing)
- Sync component libraries
- Export cards as Figma components

**CSS Variables**
- Define brand variables in CSS
- Inject into all templates automatically
- Update globally without republishing cards

## Best Practices

- Update brand assets promptly when rebranding
- Test templates across devices before publishing to team
- Use field locking to prevent accidental brand dilution
- Provide clear guidelines for custom link usage
- Review directory quarterly to remove outdated cards
- A/B test directory layouts to optimize engagement
- Keep signature designs simple for email client compatibility
