# Vulnerability Disclosure
We welcome security researchers to report vulnerabilities responsibly.
Please include detailed reproduction steps and affected components.

# Scope
- Web Application: https://cardrender.com
- App: https://app.cardrender.com
- API: https://api.cardrender.com

# Out of Scope
- Social engineering attacks
- Physical security issues
- Third-party services we don't control

# Safe Harbor
We commit to not pursue legal action against security researchers who:
- Make good faith efforts to avoid privacy violations and service disruptions
- Report issues promptly and allow reasonable time for fixes
- Do not publicly disclose vulnerabilities before they are resolved
